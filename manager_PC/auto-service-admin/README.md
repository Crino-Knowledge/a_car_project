# 汽服报价系统 - 后台管理端

基于 React 18 + Umi 4 + Ant Design 5 构建的汽服行业数字化管理平台。

## 技术栈

- **框架**: React 18 + Umi 4
- **UI组件库**: Ant Design 5.x
- **状态管理**: Dva 2.x
- **图表库**: ECharts 5.x
- **构建工具**: Webpack (Umi 内置)
- **开发语言**: TypeScript

## 功能模块

1. **登录模块** - 身份认证与安全访问控制
2. **分析页模块** - 数据概览与可视化分析
3. **工作台模块** - 运营人员工作门户
4. **用户管理模块** - 用户账户全生命周期管理
5. **店铺管理模块** - 汽服门店信息维护
6. **供应商管理模块** - 供应商信息库维护
7. **品牌管理模块** - 配件品牌库维护
8. **配件分类管理模块** - 多级配件分类体系
9. **采购管理模块** - 采购需求监控
10. **应标管理模块** - 投标行为监控
11. **订单管理模块** - 订单状态追踪

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
src/
├── pages/              # 页面组件
│   ├── Login/          # 登录页
│   ├── Dashboard/      # 分析页
│   ├── Workbench/      # 工作台
│   ├── System/         # 系统管理
│   ├── MasterData/     # 主数据管理
│   └── Business/       # 业务管理
├── components/         # 公共组件
├── models/             # Dva 模型
├── services/           # API 服务
├── utils/              # 工具函数
├── access.ts           # 权限配置
├── app.tsx             # 应用入口
└── global.less         # 全局样式
```
